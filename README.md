# Premium Responsive Contact Form (HTML/CSS/JS)

A modern, mobile-first contact form UI with inline validation, loading state, and success feedback.

## What’s included
- Responsive layout (desktop → mobile)
- Inline validation (name, email, message)
- Clear error + success states
- Loading spinner (simulated request)
- Clean, modern “glass” UI
